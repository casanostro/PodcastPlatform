🎯 Description du Projet

Malgré son nom, PodcastPlatform est une plateforme portfolio moderne avec marketplace intégré. Elle présente une esthétique terminal rétro inspirée du RobCo Terminal de Fallout 3, avec des effets CRT et une interface en ligne de commande. Le site combine un portfolio professionnel pour un expert en supply chain et transformation digitale avec une marketplace d'équipement podcast.

npm run dev
npm run build
npm run start
npm run db:push

💾 Couche de Stockage - MemStorage

Implémentation en mémoire pour le développement avec chargement depuis fichiers JSON. Interface IStorage pour abstraction et migration facile vers PostgreSQL en production.

  • Chargement automatique des données depuis projects.json et podcasts.json
  • Auto-incrémentation des IDs pour nouveaux enregistrements
  • Interface abstraite permettant le changement de backend
  • Support PostgreSQL avec Drizzle ORM configuré

🔐 Drizzle ORM

ORM TypeScript type-safe avec migrations automatiques, schemas Zod intégrés, support PostgreSQL

✅ Validation Zod

Validation côté serveur pour tous les endpoints, schémas partagés frontend/backend

🔄 Hot Reload

Développement avec tsx pour rechargement automatique, Vite HMR pour le frontend

🚀 Express.js

Serveur Node.js robuste avec middleware, support WebSocket, intégration Vite en dev
